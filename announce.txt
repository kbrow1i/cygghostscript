Subject: ghostscript 9.25-0.1

The following packages have been uploaded to the Cygwin distribution
as test releases:

* ghostscript-9.25-0.1
* libgs9-9.25-0.1
* libgs-devel-9.25-0.1

GNU Ghostscript is a PostScript interpreter capable of converting PS
files into a number of printer output formats.  Ghostscript can also
render PS files into a number of graphics file formats.

This release is based on an upstream release candidate for
ghostscript-9.25.  It should fix some bugs that were introduced in
9.24, including the one reported here:

  https://cygwin.com/ml/cygwin/2018-09/msg00162.html
